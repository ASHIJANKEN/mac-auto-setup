# mac-auto-setup
Auto setup system for macOS.

## Downloads
To get started please run:
```
git clone https://github.com/AkkeyLab/mac-auto-setup.git ~/
```

## Installation
To get started please run:
```
~/mac-auto-setup/setup.sh
```
Install GUI apps:
```
~/mac-auto-setup/app.sh
```
Install App Store apps:
```
~/mac-auto-setup/appstore.sh
```
